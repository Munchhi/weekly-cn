- [BestXTools 社区（RSS）](https://discuss-cn.bestxtools.com/t/weekly)

- [竹白（邮箱，微信）](https://bestxtools.zhubai.love/)

欢迎交流，[投稿](https://discuss-cn.bestxtools.com/d/8)，订阅。
